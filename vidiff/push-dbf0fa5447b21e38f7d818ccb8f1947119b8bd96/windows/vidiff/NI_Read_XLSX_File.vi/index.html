NI_Read XLSX File.vi

Reads all sheets from the specified <b>XLSX File</b>. This VI does <b>not</b> require Microsoft Excel to be installed.

Each sheet is returned as a 2D Array of Strings. You will need to perform additional conversion if you require data in numeric format.

Time data (date and/or time) is returned as a number of days elapsed since January 1, 1900. Hour/minute/second information will be returned as a fraction of a day (24 hours).
