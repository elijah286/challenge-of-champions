Controller.lvlib:Controller.lvclass:Pre Launch Init.vi

<p>(<strong>Filename</strong>: <span class="dnl">Actor Framework.lvlib:Actor.lvclass:Pre Launch Init.vi</span>)</p>

<p>Defines behavior that occurs after the <a href="nihelplauncher://docs/csh?context=lvcore_lvcomm_af_launch_nested_actor">Launch Nested Actor</a> method is invoked but before the actor's Actor Core method begins running.</p>

<p>By default, this method does nothing. A descendant class may override it to define behavior.</p>
